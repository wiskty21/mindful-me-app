# App Store申請タイムライン

## フェーズ1: ローカル環境で申請（推奨）

### メリット
- **コスト0円**: サーバー費用不要
- **審査中の修正可能**: ローカルで自由に変更
- **却下リスク対策**: 本番環境構築前に問題把握

### 設定方法
```javascript
// frontend/.env.production
VITE_API_URL=https://api.mindfulme.app

// 審査時は存在しないURLでOK
// PWAは静的アプリとして動作
```

## 重要な注意点

### PWAの審査ポイント
1. **オフライン動作必須**
   - Service Workerでキャッシュ
   - ネットワークエラー時の処理

2. **ネイティブ機能の活用**
   - プッシュ通知
   - バイブレーション
   - フルスクリーン

3. **App Store固有の価値**
   - 「なぜWebでなくアプリか」を説明
   - iOS限定機能をアピール
